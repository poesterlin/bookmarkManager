# BookmarkManager

A simple bookmark manager that allows you to save, organize and track your bookmarks.

[Hosted here](https://bookmarks.oesterlin.dev)

## Features

- Add, edit and delete bookmarks
- Organize bookmarks into categories
- Search bookmarks
- Tag bookmarks
- dark mode
